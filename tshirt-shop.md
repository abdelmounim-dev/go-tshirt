# Acceptance Criteria

1. Product Information
    - [ ]  I can see the product name (e.g., *Black T-shirt* or *White T-shirt*).
    - [ ]  I can see the price clearly displayed (€120).
    - [ ]  I can read a simple description that tells me what makes the shirt special (material, comfort, durability).
    - [ ]  I can see if the item is in stock and ready to ship.
    - [ ]  I can see a product image that shows me what the T-shirt looks like.
    - [ ]  The image matches the color I select.
2. Product Options
    - [ ]  I can choose between black and white colors.
    - [ ]  I can select a size (XS, S, M, L, XL, XXL).
    - [ ]  Sizes that are not available are greyed out and not clickable.
3. Add to Cart
    - [ ]  Once I pick my color and size, I can click the **“Add to cart”** button.
    - [ ]  The button is clearly visible and clickable.
    - [ ]  After adding, I get a confirmation that the item is in my cart.
4. Trust & Guarantee
    - [ ]  I can see that the product comes with a Lifetime Guarantee, which makes me feel secure in my purchase.
5. Recommendations
    - [ ]  Below the main product, I see a section called “Recommended for you”.
    - [ ]  It shows me similar products (e.g., Basic Tee in Aspen White).
    - [ ]  Each recommended product shows a price and image so I can explore more options.
